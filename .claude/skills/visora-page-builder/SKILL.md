# VISORA Page Builder

Build complete Angular pages for VISORA using the design system and Angular best practices.

## Required pages
- Login
- Forgot password
- Store list
- Store create/edit
- User assignment
- Camera dashboard
- Add/edit camera
- Camera live view
- Event list
- Event detail
- Alert detail
- Not found
- Unauthorized

## Page rules
- Each page must have clear title, purpose and main action.
- Use loading, empty, success and error states.
- Use responsive layouts.
- Use accessible labels.
- Use reusable components.
- Use mock data only when backend is not ready.
- Avoid connecting real facial data or external official databases.
