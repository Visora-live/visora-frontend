# VISORA Accessibility and Usability Review

Review the UI to help reach usability score 4.0/5 or higher.

## Check
- Responsive behavior
- Keyboard navigation
- Form labels
- Error messages
- Button clarity
- Color contrast
- Alert clarity
- Empty states
- Loading states
- Confirmation dialogs
- Table readability
- Camera grid usability

## Output
Give findings by priority:
- Critical
- Important
- Nice to have

Suggest minimal fixes first.
