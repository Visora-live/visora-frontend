# VISORA Design System

Create a modern, professional dashboard for a smart video surveillance system.

## Brand
Name: VISORA
Style: clean, secure, trustworthy, modern.

## Colors
- Background: #FBFBFC
- Secondary background: #D5DEE7
- Text: #3A414B
- Primary: #1652A3
- Accent: #2F88F5

## Semantic states
- Normal: green
- Suspicious: yellow
- Critical alert: red
- Inactive: gray

## Typography
- Lora only for logo or main brand headings.
- Use clean UI typography for forms, tables, dashboards and body text.

## UI principles
- Mobile-first responsive design.
- Clear hierarchy.
- Strong contrast.
- Large readable tables.
- Obvious primary actions.
- Alerts must be fast to understand.
- Avoid visual clutter.

- sidebar
